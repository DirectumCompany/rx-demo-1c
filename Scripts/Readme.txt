Скрипт для обновления конечных GUID'ов в ExternalEntityLink на оригинальные.
Скрипт многоразовый.

Обновление выполняется для следующих типов сущностей:
- УПД
- Счета на оплату

Требования:
Укажите корректный идентификатор системы в скриптах. Идентификатор берется из 1С → Directum RX → Настройки подключения Directum RX → Идентификатор системы.

Для MSSQL используйте скрипт -- и поменяйте идентификатор в переменной @extsystemid.
Для PostgreSQL используйте скрипт ChangeOverlapGuidToOrigin_PG и поменяйте идентификатор в переменной v_extsystemid.
